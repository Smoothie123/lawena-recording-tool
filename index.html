What's this?

Lawena is a simple Java tool which greatly enhances your TF2 and other Source games image quality for recording purposes, with surprisingly little performance loss compared to other such configurations. Records using ingame Source Recorder, with key bindings to ease the process.

Easy to install and use. This tool does not interfere with your regular TF2 configuration, HUD or launch options. Requires Steam in order to run.

Features

  • Insane max quality graphics configuration.
  • Recorded files are saved to the directory of your choosing for comfort and order.
  • Options to enable/disable various game elements which may be unnecessary in the context of a movie.
  • Includes built-in moviehuds with different levels of simplicity.
  • Supports custom HUDs/materials/models/etc to add a personal touch to your movies.
  • Includes a set of built-in skyboxes, and supports custom skyboxes via simple drag-and-drop.
  • SrcDemo2 support to reduce drive usage and space.
  • VDM file support for automatizing your movie recording process.
  • Updates automatically, you can see the latest changes here.

Installing Lawena

If you want to use a stable version, check the releases page. Simply download the zip release, extract it anywhere and run lawena.exe.

Latest versions

Lawena allows you to switch the branch to retrieve automatic updates from, just go to the Help -> Switch Updater Branch... or Help -> About menu item (depending on your version), choose one of the branches, press OK and restart the tool to apply the changes. The current branches are:

  • release-v4.1 — latest stable version (Recommended, default branch)
  • preview-v4.1 — latest updates being tested
  • release-v4.2 and snapshot-v4.2 — newer version featuring symbolic links, multiple Source game support and other features. (Unstable!)
